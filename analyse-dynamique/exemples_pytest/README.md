# Compteur

Projet qui permet de compter les caractères dans un fichier

Pour le lancer : 
```
uv sync
uv run python src/main.py
```

> Remarque vous pouvez suivre la même configuration pour qu'on ait pas nécessairement à configurer le PYTHON_PATH pour lancer votre projet

Pour les tests : 
```
uv sync --dev
uv run pytest
```
